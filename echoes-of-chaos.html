About the Game

Echoes of Chaos is a short interactive experience developed in Unity that guides players through three distinct environments, each offering its own set of challenges. From navigating treacherous pits to using light mechanics to banish ghosts, the game emphasizes tension and environmental storytelling.


Key Contributions

  • Complex AI Behavior: Programmed enemy behaviors for zombies using State Machines and Field of View detection systems to hunt the player.
  • A* Pathfinding: Implemented a formidable tank enemy in the third environment that uses the A* algorithm to intelligently navigate the level and pursue the player.
  • Robust Player Controller: Developed a single, highly customizable player controller script with parameters exposed in the Inspector for rapid iteration.
  • Interactive Mechanics: Built systems for key pickups, bomb throwing mechanics, and light-based combat interactions.
