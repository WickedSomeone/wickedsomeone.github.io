About the Game

This interactive experience is a collaborative project utilizing real-world assets created via cellphone photogrammetry. The project explores how scanned real-world data can be used to create distinctive visual styles and immersive environments.


Key Contributions

  • Photogrammetry Integration: Captured and implemented the main hallway environment using cellphone scanning software.
  • Interactive Mechanics: Scripted key events including jumping puzzles, particle avoidance challenges, and item placement logic.
  • Player Controller: Developed the primary movement and interaction scripts to navigate the scanned environment.
