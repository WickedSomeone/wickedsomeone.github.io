About the Game

Ethereal Armageddon is a vertical slice of an RPG experience where the player embarks on a quest into the wilderness. It serves as a technical demo for foundational RPG systems including combat, quests, and dialogue.


Key Contributions

  • Modular Quest System: Developed a flexible Quest and Dialogue system from scratch using Scriptable Objects, allowing for easy content expansion.
  • 3rd Person Controller: Built a custom character controller integrated with Cinemachine for smooth camera tracking and state-based movement.
  • Combat Systems: Implemented a targeting system (lock-on) and enemy AI using state machines similar to the player's logic.
  • Environment: Developed the terrain, water systems, and mountain geometry for the game world.
